# Weather Frontend M2

Proyecto frontend que muestra información de
distintas localidades, su detalle y pronóstico semanal. Desarrollado con
HTML5 semántico, Bootstrap y JavaScript.

# Características principales

HTML5 semántico en todas las páginas.
Bootstrap para diseño responsivo mobile‑first.
Listado de 10+ localidades** con navegación a página de detalle.
Pronóstico semanal dinámico.
Interacción con DOM, tarjetas, eventos y estados.
Navegación clara, accesible y usable.

# Estructura del proyecto

    /index.html      >> Home con tarjetas
    /detalle.html    >> Detalle + pronóstico
    /css/styles.css  >> Estilos organizados
    /js/app.js       >> Lógica principal
    /js/detalle.js   >> Lógica de detalle
    /assets/Backgrounds >> fondoGalaxia.jpg

# Cómo ejecutar

1- Clona el repositorio:

        git clone https://github.com/motorniko/Portafolio-M-dulo-2-APLICACIONES-FRONT-END..git

2- Abre "index.html" en tu navegador.

3- Asegura conexión a internet para que cargue Bootstrap/CDN.
